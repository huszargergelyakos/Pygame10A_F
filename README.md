# Highway Dodge

Highway Dodge is a fast-paced, top-down 2D arcade dodging game built with Python and Pygame.
You drive on an endless highway, switch lanes to survive traffic, collect coins and fuel, and use Nitro at the right moment to escape dangerous situations.

The goal is simple: stay alive as long as possible, travel the greatest distance, and beat your best score while the game gets faster and more intense over time.

## Features
- Object-Oriented Programming (OOP) architecture with separated game systems
- Clean, modular code structure for easier development and maintenance
- 4-lane movement system (Subway Surfers style)
- Endless scrolling road and dynamic gameplay pacing
- Coin collection and score tracking system
- Fuel pickups and Nitro gauge with burst speed mechanic
- Car selection screen with multiple playable car options
- Biome progression system as distance increases
- Multiple obstacle and enemy vehicle types
- In-game HUD with distance, coins, time, biome, and Nitro status

## Controls
- **Left Arrow / A**: Move one lane to the left
- **Right Arrow / D**: Move one lane to the right
- **Space**: Use Nitro (Speed boost)
