# Highway Dodge

A fast-paced, top-down 2D dodging game built with Python and Pygame. Navigate through highway traffic, collect coins and fuel, and avoid incoming obstacles to get the highest score possible!

## Features
- Object-Oriented Programming (OOP) architecture
- Clean code principles with modular design
- 4-lane movement system (Subway Surfers style)
- Endless scrolling background
- Collectibles and score system

## Controls
- **Left Arrow / A**: Move one lane to the left
- **Right Arrow / D**: Move one lane to the right
- **Space**: Use Nitro (Speed boost)
